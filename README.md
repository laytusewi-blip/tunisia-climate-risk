# Tunisia Climate Risk Assessment System

An AI-powered climate risk analysis and insurance premium calculator for Tunisia neighborhoods.

## Features
- Interactive risk assessment for flood, wildfire, hurricane, and drought hazards
- Machine learning-based risk scoring
- Insurance premium calculation
- Interactive map visualization
- Real-time property analysis

## Tech Stack
- Python
- Streamlit
- GeoPandas
- Folium (maps)
- Plotly (charts)
- XGBoost (ML)
- Scikit-learn

## Live Demo
https://tunisia-climate-risk-lch9axvcifzbywgld9jgfl.streamlit.app/

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```
